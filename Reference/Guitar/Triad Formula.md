# Triad Formula

1. **Major Triad:**
   - Formula: 1, 3, 5

2. **Minor Triad:**
   - Formula: 1, b3, 5

3. **Diminished Triad:**
   - Formula: 1, b3, b5

4. **Augmented Triad:**
   - Formula: 1, 3, #5

5. **Dominant 7th Chord:**
   - Formula: 1, 3, 5, b7

6. **Major 7th Chord:**
   - Formula: 1, 3, 5, 7

7. **Minor 7th Chord:**
   - Formula: 1, b3, 5, b7

8. **Half-Diminished 7th Chord:**
   - Formula: 1, b3, b5, b7

9. **Diminished 7th Chord:**
   - Formula: 1, b3, b5, bb7 (double flat 7)

For extended chords:

10. **9th Chord:**
    - Formula: 1, 3, 5, b7, 9

11. **11th Chord:**
    - Formula: 1, 3, 5, b7, 9, 11

12. **13th Chord:**
    - Formula: 1, 3, 5, b7, 9, 11, 13
