# The cult of done


![[3327377382_be705a4089_o 1.png]]



1. There are three states of being: Not knowing, action , completion
2. Everything is a draft
3. There is no editing stage
4. Pretending to know what you're doing is the same as knowing 
	1. Banish procrastionation. If you wait more than a week to get it done banish it
5. The point of being done is not to finish but to get other things done
6. once youre done you can throw it away
7. laugh at perfection , its boring.
8. people without dirty hands are wrong, doing 'something' makes you right
9. failure counts as done so make mistakes
10. destruction is a variant of done
11. If you have an idea publish it on the internet it counts as done
12. done is the engine of more
